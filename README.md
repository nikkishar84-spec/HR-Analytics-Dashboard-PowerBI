# 📊 HR Analytics Dashboard | Power BI

An interactive **HR Analytics Dashboard** built using **Power BI, Excel, and SQL** to analyze employee data, monitor key HR metrics, and identify workforce trends. The dashboard provides valuable insights into employee attrition, demographics, job roles, and departmental performance through interactive visualizations.

---

## 📌 Project Overview

This project demonstrates how HR data can be transformed into meaningful business insights using Power BI. The dashboard helps HR professionals understand employee distribution, analyze attrition, and make data-driven decisions for workforce planning.

---

## 📈 Key Performance Indicators (KPIs)

- 👥 Total Employees
- 📉 Attrition Rate
- 🎂 Average Age
- 💰 Average Salary

---

## 📊 Dashboard Features

- Employees by Department
- Gender Distribution
- Employees by Age Group
- Top 7 Job Roles
- Attrition by Overtime
- Interactive Filters
  - Department
  - Gender
  - Attrition

---

## 💡 Business Insights

- Research & Development has the highest number of employees.
- Employees working overtime show higher attrition than those who do not.
- The dashboard helps identify workforce distribution across departments and job roles.
- HR teams can use these insights to support employee retention and workforce planning.
